## Resource

$ nest g resource name

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Migrations

```bash
# auto generate
$ npm run migration:generate

# apply
$ npm run migration:up

# rollback
$ npm run migration:down
```
